# 单调栈 monotonic stack  
>从左向右遍历需要在其中寻找特定值的那个数组.    
遍历过程中把不满足的pop(-1), 维持栈顶`st(-1)`就是满足条件的那个值
即使去掉无用数据，保证栈中数据有序
